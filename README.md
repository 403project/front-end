# 별별투표 프로젝트

## 기술 스택

- Next.js 14
- React-Query
- Typescript
- vanilla extract css
- Eslint
- Prettier
- Zustand

## git 방식

- master, develop, feature 브랜치 사용합니다.

## git 커밋 메시지

- {type} : {subject}
- feat : 새로운 기능 추가
- fix : 버그 수정
- refactor : 코드 리팩토링
- docs : 문서 수정
- style : 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
- test : 테스트 코드, 리팩토링 테스트 코드 추가
- chore : 빌드 업무 수정, 패키지 매니저 수정
- etc : 그 외 자잘한 수정
